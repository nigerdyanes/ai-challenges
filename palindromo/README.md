# Reto:

Escribe una función llamada es_palindromo que tome una cadena de texto como entrada y devuelva True si la cadena es un palíndromo y False si no lo es. Un palíndromo es una palabra, frase, número o cualquier otra secuencia de caracteres que se lee igual hacia adelante que hacia atrás, ignorando los espacios, mayúsculas y minúsculas. Por ejemplo, "reconocer" es un palíndromo.
